# 🚀 Twelve Data Integration - הגדרה מדריך

## מה זה Twelve Data?
**Twelve Data** הוא מקור נתונים פיננסיים מהשורה הראשונה בעולם, זהה ל-Bloomberg ו-Reuters!

- ✅ **נתונים בזמן אמת** (real-time quotes)
- ✅ **תקצוב חינם** (יום ראשון בחודש - חינם לחלוטין)
- ✅ **כיסוי עולמי** (מניות, סחורות, קריפטו, טלאס)
- ✅ **API מהימן** (שימוש בתעשיה)

---

## 📋 שלבי ההגדרה

### שלב 1: קבל את ה-API Key
1. הכנס לאתר: **https://twelvedata.com**
2. לחץ **"Sign Up"** (צפוי חשבון חינם)
3. אמת את המייל שלך
4. גש ל- **Dashboard → API Keys**
5. **העתק את ה-API Key**

### שלב 2: הוסף למשתנים הסביבה ב-Render
1. גש לפרויקט שלך ב-Render
2. **Settings** → **Environment**
3. לחץ **Add Environment Variable**
4. **Key:** `TWELVE_DATA_API_KEY`
5. **Value:** [הקוד שהעתקת]
6. לחץ **Save**
7. **Deploy** מחדש

### שלב 3: בדוק שהכל עובד
כשהאפליקציה תעלה, פתח את הטאב **"📡 נתונים חיים"** ובדוק ש-AAPL מראה מחיר!

---

## 🔄 זרימת Fallback

```
Twelve Data ✅ (מהיר + מדויק)
    ↓ (אם לא זמין)
Finnhub ✅ (מהיר יחסית)
    ↓ (אם לא זמין)
yfinance ⚠️ (איטי אבל תמיד עובד)
```

---

## ✅ בדיקה סופית

בתוך האפליקציה, בטאב "📡 נתונים חיים":
- **"Twelve Data 🟢 חי"** = הכל עובד מושלם! 🎉
- **"finnhub 🟢 חי"** = Twelve Data לא זמין, בשימוש Finnhub (טוב בכל זאת)
- **"yfinance"** = שניהם לא עבדו, fallback לישן (איטי אבל חוקי)
